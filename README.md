
# しょぼちゃっと

## Overview
シンプルなチャットアプリです

## Feature
- WebSocket
    - 他のユーザーが送った後でも即座に反映される
- Markdown
    - 見やすいメッセージを送ることができる

## Requirement
Node.js v10.15.3 以上

## Usage
以下を行い， http://localhost:8000 にアクセスすると，チャットアプリを使用できます．
```
git clone https://github.com/tatsuya-f/shobochat.git
cd shobochat
npm install
npm run gulp
npm start
```
